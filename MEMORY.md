# Workflow

## Issue Workflow
1. Create an issue branch from `main`: `git checkout -b <type>/<short-description>`
2. Work on changes in the branch
3. Commit changes with descriptive messages
4. Push the branch: `git push origin <type>/<short-description>`
5. Create a PR: `gh pr create --title "<title>" --body "<description>"`
6. After review, merge the PR
