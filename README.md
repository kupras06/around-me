# AroundMe

**AroundMe** is a mobile-first place discovery app that shows a curated map of quality cafés, restaurants, stores, and experiences — **recommended by creators you trust**.

Unlike traditional discovery platforms (crowd-sourced star ratings, anonymous reviews, and paid rankings), AroundMe makes the *source of taste* the primary interface: every pin is tied to a real creator, with their note front and center.

## MVP scope
- **Launch city:** Bengaluru, India  
- **Neighbourhoods:** Indiranagar + Koramangala  
- **Core flow:** Map → tap pin → creator note → save/share/directions  
- **Creator onboarding:** Instagram/Twitter OAuth + verification tiers based on follower count  
- **No ratings, no reviews, no sponsored pins**

## Tech (planned)
- React Native (Expo)
- Mapbox
- Supabase (Postgres, Auth, Storage, Edge Functions)

*A map of good things — curated by people you trust.*
