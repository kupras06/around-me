# AroundMe

*A map of good things — curated by people you trust.*

AroundMe is a mobile-first place discovery app that helps people find great cafés, restaurants, stores, and experiences through **creator-curated recommendations**. Instead of crowd-sourced star ratings and algorithmic rankings, every place pin is backed by a **specific, identifiable creator** the user chooses to follow—making the trust layer explicit and visual on a map.

**MVP launch:** Bengaluru, India (Indiranagar + Koramangala)  
**Platforms:** iOS + Android  
**Status:** In planning (MVP)

---

## What makes it different

- **Trust-first discovery:** pins are recommendations from creators, not anonymous averages.
- **Creator identity is always visible:** every pin has a face/name behind it.
- **No noise:** no star ratings, no review counts, no sponsored pins.
- **Map-first experience:** the map is the product; everything else supports it.

---

## MVP Features

### Map
- Full-screen map experience
- Category filters: **All · Cafés · Diners · Stores · Experiences**
- Pins are color-coded by category
- Pins from followed creators show a distinct highlight
- Pin clustering at higher densities
- Neighbourhood scope selector (MVP: Indiranagar, Koramangala)

### Place details
- Place name, category, neighbourhood
- Creator attribution: avatar, name, verification tier, follower count
- Creator note (up to **160 characters**) as the focal content
- If multiple followed creators pinned the same place, users can view each note
- Actions: **Save · Share · Directions** (opens Google/Apple Maps)

### Save & revisit
- Saved places list (grouped by category)
- Toggle to view saved places on a saved-only map
- Each saved entry keeps creator attribution

### Creators
- Browse creators relevant to the launch neighbourhoods
- Creator cards: avatar, handle, follower count, verification badge, focus description, pin count
- Follow/unfollow with minimal friction
- Filters: **Food · Lifestyle · Travel · Local**

### Creator profiles
- Creator bio + focus
- All pins by that creator shown as a list
- Tap a pin → open place details

### Creator onboarding + pinning
- Instagram or Twitter OAuth
- Follower count pulled automatically → verification tier assigned
- Creators can submit pins with:
  - Google Places autocomplete (to prevent duplicates)
  - Category selection
  - Note (160 chars)
  - Optional photo upload

### Submissions & moderation (MVP)
- Verified creators: pins auto-approved
- “Trusted Local” creators: pins go to a manual admin review queue

---

## Creator verification tiers (MVP)

- **Verified (10,000+ followers):** auto-approved pins
- **Trusted Local (1,000–9,999):** manual review queue
- **Community (<1,000):** post-MVP

- **City:** Bengaluru

## Success metric (north star)

**Visit rate:** % of users who save a place and visit within 14 days (target: **30%**).
